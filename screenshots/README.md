# Screenshots for the main README

Add **PNG** (or **JPG**) files here so the images in the root `README.md` render on GitHub.

| Filename        | What to capture |
| --------------- | ---------------- |
| `overview.png`  | Full page: sidebar controls + graph canvas |
| `running.png`   | Mid-animation: highlighted node / relaxed edge |
| `final-path.png`| Done state: green shortest path + Path & Cost text |

**Tip:** Use a wide window (e.g. 1280–1440px), capture the browser chrome if you want, or crop to the page only.

After saving, commit the images:

```bash
git add screenshots/*.png
git commit -m "Add README screenshots"
```
